SMR - Informática aplicada a Sistemas Electrónicos (Robótica) y Programación en Python

Descripción

En esta materia, aprendemos a programar en Python y cómo aplicar estos conocimientos a la robótica. Se incluyen tareas prácticas sobre la programación de robots y la integración de sensores.

Trabajos Realizados

  • Proyecto: Control de un robot móvil usando Python y sensores de distancia.
  • Proyecto: Desarrollo de un programa en Python para simular el movimiento de un brazo robótico.

Notas/Exámenes

Examen parcial sobre fundamentos de robótica: 80%

Nota final esperada: 90%
